# COMP3011 Fitness API

A RESTful Fitness Habit & Workout Tracking API built with **FastAPI** and **SQLite**.  
The API allows users to log workouts, manage workout history, and generate simple analytics such as workout streaks and weekly summaries.

This project was developed for the **COMP3011 – Web Services and Web Data** module at the University of Leeds.

---

# Features

• Full **CRUD operations** for workout logs  
• **Analytics endpoints** including workout streaks and weekly summaries  
• **SQLite database integration** using SQLAlchemy  
• **RESTful API design** with proper HTTP status codes  
• **Automatic interactive API documentation** via Swagger UI  
• **Automated tests** using pytest  

---
